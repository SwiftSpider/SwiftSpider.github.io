Variables

you can make variables by this command
var (name) = (value)
you can make them words or "strings" by adding quotes to it
var (name) = "(word)"

Text

once you add a piece of text in html you can change it with and "Id"
go behind the FIRST greater than symbol on the tag of the text and type id="test" or whatever you want it to be called

<p id="test">text</p>

on a js file ex.(script.js)

document.getElementById(test).innerHTML = "text";
